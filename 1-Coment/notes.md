Komentar di JavaScript digunakan untuk menulis catatan di dalam kode yang tidak akan dijalankan oleh program. Fungsinya antara lain:

Menjelaskan kode: Membantu programmer lain (atau diri sendiri di masa depan) memahami maksud kode.
Menonaktifkan kode sementara: Bagian kode bisa dikomentari agar tidak dijalankan saat debugging.
Membuat kode lebih rapi dan mudah dipelihara.
Ada dua jenis komentar di JavaScript:

Komentar satu baris:
Menggunakan  // di awal baris.

Komentar banyak baris:
Menggunakan /* ... */.

Komentar sangat penting untuk kolaborasi dan pemeliharaan kode.
